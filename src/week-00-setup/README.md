# Week 0: 环境搭建

> 工欲善其事，必先利其器

## 📋 本周目标

- ✅ 安装 GHCup 和 Haskell 工具链（GHC、Cabal、HLS）
- ✅ 配置 VS Code 开发环境
- ✅ 学会使用 GHCi 交互式解释器
- ✅ 运行你的第一个 Haskell 程序
- ✅ 排查常见安装问题

## ⏱️ 预计时长

- **安装时间**：30-60 分钟（取决于网络速度）
- **学习时间**：1-2 小时
- **练习时间**：30 分钟

## 📚 学习材料

1. **[详细讲义](lecture.md)** - 逐步安装和配置指南
2. **[练习作业](exercises.md)** - 实践练习

## 🎯 前置要求

- ✅ Windows 10/11、macOS 10.13+、或现代 Linux 发行版
- ✅ 至少 5 GB 可用磁盘空间
- ✅ 稳定的网络连接
- ✅ 基本的命令行使用能力

## ⚠️ 常见问题

如果安装过程中遇到问题，请参考讲义中的"故障排查"部分，或在 GitHub Issues 中提问。

### 快速问题检查清单

- [ ] 用户名和安装路径没有中文或空格（Windows 用户特别注意）
- [ ] 已安装系统命令行工具（macOS 需要 Xcode Command Line Tools）
- [ ] 防火墙允许 GHCup 下载文件
- [ ] 环境变量已正确设置（重启终端后生效）

## 💡 自学建议

### 如果遇到困难

1. **不要跳过安装步骤** - 每一步都很重要
2. **多试几次** - 网络问题很常见，重试通常能解决
3. **寻求帮助** - 在 [GitHub Issues](https://github.com/mauyin/haskell-101-cn/issues) 提问
4. **搜索错误信息** - 将错误信息复制到 Google/Stack Overflow

### 学习节奏建议

- **不要着急** - 第一次接触函数式编程会感觉陌生，这很正常
- **多实验** - 在 GHCi 中随意尝试，不用担心弄坏什么
- **记录问题** - 把不理解的地方记下来，后面会逐渐清晰
- **每天 30 分钟** 比周末突击 5 小时效果更好
