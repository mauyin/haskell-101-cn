# Week 0 练习：环境搭建验证

完成这些练习来验证你的 Haskell 环境搭建是否正确。

## 练习列表

### 练习 1: Hello Haskell
**文件**: `exercise-01-hello.hs`

编写基础的 Haskell 函数，熟悉函数定义和类型签名。

**难度**: ⭐☆☆☆☆

**目标**:
- 理解函数定义语法
- 学会编写类型签名
- 在 GHCi 中加载和测试函数

### 练习 2: GHCi 操作练习
**文件**: `exercise-02-ghci.md`

通过一系列 GHCi 命令熟悉交互式环境。

**难度**: ⭐☆☆☆☆

**目标**:
- 掌握 GHCi 基本命令
- 学会查看类型和信息
- 能够在 REPL 中实验代码

## 如何完成练习

### 对于 .hs 文件

1. 在文件中完成函数实现
2. 在 GHCi 中加载文件：
   ```haskell
   ghci> :load exercise-01-hello.hs
   ```
3. 测试你的函数
4. 对照 `solutions/` 目录中的参考答案

### 对于 .md 文件

1. 按照文件中的指示操作
2. 在终端中实际运行命令
3. 记录输出结果
4. 对照参考答案检查

## 评分标准

- ✅ **功能正确** (60%) - 函数按要求工作
- ✅ **类型签名** (20%) - 所有函数都有正确的类型签名
- ✅ **代码风格** (10%) - 代码清晰易读
- ✅ **测试完整** (10%) - 在 GHCi 中测试过所有函数
