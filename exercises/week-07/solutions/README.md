# Week 7 Solutions

本目录包含 Week 7 所有练习的参考答案。

## 文件说明

- `Week07Exercises-Solutions.hs` - Exercise Set 1, 2, 4 的完整答案
- `Week07API-Solutions.hs` - Exercise Set 3 (API) 的完整答案
- `balance-checker-complete/` - 项目 1 的完整实现
- `tx-explorer-complete/` - 项目 2 的完整实现

## 使用建议

1. **先自己尝试**：在查看答案前，尽量自己完成练习
2. **对比学习**：完成后对比答案，学习不同的实现方式
3. **理解思路**：重点理解解题思路，而不是死记代码
4. **扩展练习**：在答案基础上尝试添加新功能

## 注意事项

- 答案仅供参考，可能有多种正确实现
- 某些函数使用了简化实现，真实项目可能需要更复杂的逻辑
- API 相关代码需要有效的 Blockfrost API Key 才能运行

## 学习建议

### 对于 JSON 解析练习
- 重点理解 aeson 的 FromJSON 类型类
- 学习 parseJSON 的各种模式
- 掌握错误处理

### 对于地址操作练习
- 理解 Cardano 地址格式
- 学习字符串处理技巧
- 掌握类型识别逻辑

### 对于 API 练习
- 理解 req 库的使用
- 学习 HTTP 请求模式
- 掌握错误处理和重试逻辑

### 对于交易构建练习
- 理解 UTXO 选择算法
- 学习费用计算方法
- 掌握交易平衡逻辑

## 项目完整实现

你可以：
